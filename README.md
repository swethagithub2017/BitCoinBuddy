## Project Description

BitCoin Buddy is an app utlizing React.js allowing user to set up SMS notification for user defined BitCoin price changes. 

## Technologies Used

* React.js
* HTML
* CSS
* JavaScript
* Body-parser
* Bootstrap
* Node.js
* Chart.js
* Express
* MongoDB
* Mongoose
* Cron
* Morgan
* Request
* Twilio
